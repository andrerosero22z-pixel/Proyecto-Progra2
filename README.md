# Sistema de Gestión de Inventario para Tienda

Proyecto académico de **Programación 2** desarrollado en **Java** con enfoque en **Programación Orientada a Objetos (POO)**.

El sistema está basado en un diagrama UML que incluye clientes, proveedores, productos, pedidos, órdenes de compra e inventario.

## Objetivo

Desarrollar una aplicación de escritorio que permita gestionar el inventario de una tienda, registrando productos, clientes, proveedores, pedidos y órdenes de compra.

## Tecnologías

- Java 17
- Java Swing
- Maven
- MySQL
- Git y GitHub

## Módulos principales

- Gestión de clientes
- Gestión de proveedores
- Gestión de productos
- Gestión de inventario
- Registro de pedidos de clientes
- Registro de órdenes de compra a proveedores

## Cómo ejecutar

1. Abrir el proyecto en NetBeans, IntelliJ IDEA o VS Code.
2. Verificar que Java 17 esté instalado.
3. Ejecutar la clase:

```text
ec.edu.inventario.app.Main
```

## Base de datos

El archivo SQL inicial está en:

```text
database/schema.sql
```

Primero se debe crear la base de datos en MySQL ejecutando ese script.
